## 🖼️ How to Add Real Project Images

> Replace the placeholder images above by following these 3 steps:

**Step 1 — Create an `assets/` folder in your profile repo**
```
ranasheikh64/
  └── assets/
      ├── ecommerce-banner.png      ← 900 × 400 px
      ├── social-banner.png
      ├── fitness-banner.png
      └── task-banner.png
```

**Step 2 — Take a good screenshot of your app (Android Studio / Device)**
```bash
# Using ADB to capture a screenshot from connected device:
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png ./assets/ecommerce-banner.png
```

**Step 3 — Replace the placeholder line in this README**
```markdown
<!-- Remove this line: -->
<img src="https://placehold.co/900x400/..." />

<!-- Add this line with your file: -->
<img src="https://raw.githubusercontent.com/ranasheikh64/ranasheikh64/main/assets/ecommerce-banner.png" width="90%" />
```

> ✅ **Recommended image sizes:**
> | Type | Width × Height | Format |
> |------|---------------|--------|
> | Project Banner | 900 × 400 px | PNG / WEBP |
> | App Screenshot | 400 × 860 px | PNG |
> | Multiple Screenshots Side-by-Side | Combine in Figma/Canva first | PNG |
